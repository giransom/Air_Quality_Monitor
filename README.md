# Air_Quality_Monitor
This project is focused on building a mobile air quality monitor which can measure levels from any location. This project utilizes an Arduino ESP32, an Air Quality Sensor, and a OLED Screen with wireless monitoring. Future expansion on this project is to make a PCB that includes the temperature and air quality sensor in one.
